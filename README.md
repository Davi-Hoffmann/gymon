## Projeto GymON 

Sistema de Cadastro e Login de alunos e personal trainers, onde cada personal pode registrar o treino para cada aluno cadastrado seguir, de forma personalizada.

## Ferramentas utilizadas:

* HTML

* CSS

* PHP

* MYSQL

* JAVASCRIPT
